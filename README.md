# 🎯 Guess My Number Game

A simple and fun JavaScript game where the player tries to guess a secret number between 1 and 20.

(You can add your live link here later)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🎮 How to Play
1. Enter a number between 1 and 20.
2. Click the "Check" button.
3. The game will tell you if your guess is too high or too low.
4. Try to guess the correct number with the highest score!

## 📂 Project Structure
- index.html
- style.css
- script.js

## 💡 Features
- Random secret number generation
- Score tracking
- Highscore saving
- Reset button
